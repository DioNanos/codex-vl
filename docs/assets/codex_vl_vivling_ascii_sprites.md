# codex-vl Vivling CRT Sprites

Hand-authored ASCII CRT sprites for the compact 3-line terminal strip.

Rules:
- ASCII only
- Each sprite is exactly 3 lines tall
- Width target: 7-16 chars
- Designed for narrow terminal CRT strip
- Preserve spaces

---
