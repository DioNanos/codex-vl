Level, spawn, save, and move rules

Level-60 spawn rule

  • a species is not spawnable until its first instance reaches level 60
  • the first level-60 instance unlocks one additional spawnable instance
  • each additional level-60 instance unlocks one more spawnable instance

Save and move rules

  • moving a Vivling must preserve identity, memory, and work history
  • spawned instances retain lineage back to the anchor instance
  • legendary Vivlings use the same save model

Minimum portable fields

instance_id, species_id, species_name, family, role, rarity, level, xp, unlock flags, spawn capacity, learned work types, work preferences, memory summary, memory payload, work history, assignment status, origin version, last updated timestamp.
