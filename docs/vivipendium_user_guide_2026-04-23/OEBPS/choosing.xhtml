Which Vivling to use for which work

  • Builder: scaffolding, implementation structure, system layout, feature assembly.
  • Researcher: discovery, comparison, option analysis, investigation.
  • Reviewer: critique, QA, policy, alignment, risk validation.
  • Operator: execution, runbooks, repeated tasks, throughput jobs.
  • Bud: first-pass and foundational work.
  • Shell: durable infrastructure and defensive work.
  • Spark: fast iteration and energetic loops.
  • Bloom: visual, expressive, and taste-sensitive work.
  • Prism: technical analysis and precision work.
  • Crest: authority, judgment, and prioritization.
  • Weaver: orchestration, connection, and multi-part systems.
  • Shade: subtle diagnostics and edge-case handling.

Default selection order

  1. choose the role: build, research, review, or operate
  2. choose the family temperament
  3. check whether the species already has unlocked spawn capacity
  4. only then start parallel work
