# Codex VL 0.132.0

Codex VL 0.132.0 updates the fork to the upstream Codex 0.132.0 release line
and keeps the existing fork packaging and install paths aligned with
`@mmmbuto/codex-vl`.

## Upstream Codex 0.132.0

- Python SDK auth supports API key login, ChatGPT browser and device-code flows,
  account inspection, and logout APIs.
- Python turn APIs are easier to use for text-only workflows and handle-based
  runs now return richer turn results.
- `codex exec resume` supports `--output-schema` for structured output on
  resumed sessions.
- TUI startup capability probes are batched before the first interactive frame.
- Remote executor registration can use standard Codex auth.
- App-server turns preserve requested image fidelity across user inputs and
  image-producing tools.

## Packages

- Linux x64 native npm package.
- Termux Android arm64 native npm package.
- macOS arm64 source-build npm package.
